# 🎯 Random Name Picker

A fun and fair way to randomly select students in your classroom.

## 🌐 Live Demo
https://dryck.github.io/random-picker

## ✨ Features

- 🎡 **Visual Spinning Wheel** - Engaging animation that students love
- ⚖️ **Equity Mode** - Ensures everyone gets picked once before anyone gets picked twice
- 🔊 **Sound Effects** - Audio feedback when spinning and when name is selected
- 📱 **Mobile Responsive** - Works on teacher laptops, tablets, and phones
- 💾 **No Setup Required** - Single HTML file, works offline
- 🎉 **Celebration Effects** - Confetti animation when name is selected

## 🎯 Why Equity Mode Matters

Research shows that teachers unconsciously favor certain students when calling on them. Equity Mode helps ensure:

- All students get equal opportunities to participate
- Quiet students aren't overlooked
- No one gets called on twice while others wait

> "The equity feature has completely changed how I run discussions. Every student knows they'll get their turn." — *Elementary Teacher*

## 📖 How to Use

### 1. Add Your Students
Paste your class list into the text area, one name per line:
```
Alice
Bob
Charlie
Diana
Evan
```

### 2. Choose Your Mode
- **Equity Mode ON** (recommended): Everyone gets picked once before repeats
- **Equity Mode OFF**: Pure random selection

### 3. Spin the Wheel
Click "🎲 Spin" and watch the wheel animate!

### 4. See the Results
The selected name appears with a celebration effect. Picked students are tracked below.

### 5. Reset When Needed
Click "🔄 Reset" to start fresh with the same class.

## 💡 Classroom Tips

**For Cold Calling:**
- Use Equity Mode to ensure fair participation
- Students stay engaged knowing they might be called

**For Group Formation:**
- Spin multiple times to create random groups
- Helps mix ability levels and social groups

**For Special Roles:**
- Line leader, paper passer, today's helper
- Makes mundane tasks feel special

**Building Culture:**
- "Wheel of Praise" - spin to give compliments
- Celebrate when quiet students participate

## 🔒 Privacy

All data stays in your browser. No names are sent to any server.
